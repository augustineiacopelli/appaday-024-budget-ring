# 024 — Budget Ring

**AppADay #024** · Category: Data Visualization

A spending visualizer that renders up to 8 budget categories as a live donut chart with percentage labels. Enter category names and amounts, watch the ring update instantly, and email yourself a plain-text report with one tap.

## Live App

[augustineiacopelli.github.io/appaday-024-budget-ring](https://augustineiacopelli.github.io/appaday-024-budget-ring)

## Features

- Animated donut chart with color-coded segments and gap spacing
- Up to 8 categories, each with a name and dollar amount
- Live total display in the center of the ring
- Animated legend with percentage breakdowns
- Settings modal (⚙️) to save a report email address via localStorage
- Email Report button opens a native mailto: with a formatted budget summary
- Fully responsive — works on 375px mobile viewports
